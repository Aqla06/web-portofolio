Project Saya

Sistem Pengontrol Pakan Ayam Aduan Menggunakan Sensor HX711 Di Arduino Uno Berbasis IoT

Sistem ini menggunakan sensor berat HX711 untuk mengukur jumlah pakan secara real-time dan dikontrol oleh Arduino Uno. Data dikirim melalui teknologi IoT sehingga pengguna dapat memantau dan mengontrol pemberian pakan dari jarak jauh. Fitur utama meliputi pengontrolan jumlah pakan melalui aplikasi android dan sistem akan mendapatkan sinyal lalu menjalankan alat dan akan berhenti ketika sensor HX711 membaca berat sesuai dengan data yang dikirim dari aplikasi android. Proyek ini membantu meningkatkan efisiensi peternakan dan mengurangi pakan yang berlebih.

Rancang Bangun Aplikasi Mobile Geo Tagging Untuk Pelaporan Masalah Layanan PDAM Menggunakan Metodelogi Prototyping (Study Kasus Kunjorowesi)

Aplikasi Mobile Geo Tagging untuk Pelaporan Masalah Layanan PDAM adalah sistem berbasis mobile yang dikembangkan untuk meningkatkan efektivitas pelaporan gangguan layanan air oleh masyarakat. Aplikasi ini memanfaatkan teknologi GPS untuk menentukan lokasi secara otomatis (geo tagging), serta menyediakan fitur unggah foto dan deskripsi masalah secara real-time. Data laporan dikirim ke server dan dapat dipantau oleh admin melalui sistem backend. Pengembangan aplikasi menggunakan metodologi prototyping, yang memungkinkan proses iterasi berdasarkan umpan balik pengguna sehingga menghasilkan sistem yang lebih sesuai kebutuhan. Hasil dari proyek ini menunjukkan peningkatan kecepatan pelaporan, akurasi lokasi, serta kemudahan monitoring bagi pihak pengelola layanan.

KLASIFIKASI KUALITAS DAUN SEHAT DAN TIDAK SEHAT PADA TANAMAN JAGUNG DENGAN MENGGUNAKAN METODE MORFOLOGI GRADIEN DI PYTHON

Proyek ini mengembangkan sistem klasifikasi citra untuk mendeteksi kualitas daun jagung menggunakan metode morfologi gradien berbasis Python. Proses dimulai dengan pengolahan citra (preprocessing) seperti grayscale dan filtering, kemudian dilanjutkan dengan ekstraksi fitur menggunakan teknik morfologi gradien untuk menonjolkan perbedaan struktur tepi antara daun sehat dan tidak sehat. Hasil analisis digunakan untuk mengklasifikasikan kondisi daun secara otomatis. Sistem ini dapat membantu petani dalam melakukan deteksi dini terhadap penyakit tanaman sehingga meningkatkan efisiensi pemantauan dan potensi hasil panen.
